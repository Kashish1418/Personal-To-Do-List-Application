# Personal To-Do List Application
**Company:** VaultOfCodes
**Author:** Kashish Gupta (Python Intern)

Two implementations:
- **CLI version**: `todo_cli.py`
- **GUI version**: `todo_gui.py`

Both share the same `tasks.json` file.

## Run CLI
```bash
python todo_cli.py
```

## Run GUI
```bash
python todo_gui.py
```

## Features
- Add, view, edit, delete tasks
- Mark completed
- Categorization
- JSON storage
